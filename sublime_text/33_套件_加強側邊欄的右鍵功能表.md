# SideBarEnhancements
加強側邊欄右鍵功能表


## 安裝套件
1. 設定 > Package Control(或使用 ctrl + shift + p)
2. 輸入 Install Package 連線網路後跳出套件查詢視窗
3. 輸入 SideBarEnhancements



## 錯誤 出現找不到 wt.ext
右鍵執行 open / run 出現找不到 wt.ext
請使用micotsoft store 下載 windows ternminal
